## Description

PureFlow is a portable VSCodium environment for developers who want to keep manual coding, debugging, API recall, and explanation skills while still using AI elsewhere. AI can frame a focused Rep before work and run a Senior Defense afterward, but generation is outside the loop during Pure Mode. The product records a local, privacy-safe process trail and produces a Rep Card with real test, debug, source, and timing evidence.

Completed Reps can optionally publish a commitment to `RepRegistry` on Monad Testnet. The contract stores only a hash and public aggregate counters. Code, filenames, goals, outcomes, and notes remain local. A public companion performs a real read against the registry and reports unconfigured, missing, or verified states without hardcoded success.
